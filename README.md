# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

## Running the project locally

1.  **Install dependencies:**
    Open your terminal and run the following command to install all the necessary packages for the project.
    ```bash
    npm install
    ```

2.  **Run the development server:**
    Once the dependencies are installed, start the Next.js development server.
    ```bash
    npm run dev
    ```
    This will start the application, which you can typically view at [http://localhost:9002](http://localhost:9002).

3.  **Run Genkit (for AI features):**
    If your project uses generative AI features with Genkit, you'll need to run it in a separate terminal.
    ```bash
    npm run genkit:dev
    ```
